# Human-Computer Interaction Notes

This repository contains structured notes for the **Human-Computer Interaction (HCI) 2025/2026** course.

The material is organized to make navigation simple and to keep documents, modules, and media consistently structured.

## Repository Organization

Notes follow a hierarchical structure:

Theme → Module → Document → Media

- **Themes** represent major conceptual areas of the course
- **Modules** group related lessons
- **Documents** contain the lecture notes
- **Media folders** store the images used inside the document

For detailed naming rules and contribution guidelines see:

[CONTRIBUTING.md](CONTRIBUTING.md)
